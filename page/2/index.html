SQL注入

SQL注入

sql注入介绍

传入非加密数据的地方可能存在注入点

什么是SQL注入

  • sql注入也是正常的web端口访问
  • 只是转入的参数值并非程序设计者所希望的，而是传入了嵌套SQL代码的参数值
  • 参数值利用程序处理注入者的逻辑，按注入者的期望执行数据库查询
  • 甚至页面呈现按SQL注入者期望的显示
  • 通过应用程序把带有SQL代码的参数传递给数据库引擎

SQL注入漏洞的根本原因

  • 应用程序的问题而不是系统和数据库的问题

文件上传漏洞

文件上传漏洞

文件上传与下载漏洞的业务场

文件上传漏洞介绍

文件上传漏洞是攻击利用文件上传部分的控制不足或者处理缺陷，越过自身权限上传动态脚本文件。上传文件可以是木马、病毒、恶意脚本、webshell等。这种攻击时最直接有效的。文件上传本身没有问题，问题是服务器如何处理上传上来的文件。

webshell是以asp、php、jsp或者cgi等网页文件形式存在的一种命令执行环境，也可称为网页后门。攻击者利用浏览器访问这些后门，得到命令执行环境，以达到控制网站服务器的目的。

ICMP

ICMP

ICMP 的全称是 Internet Control Message Protocol(互联网控制协议)，它是一种互联网套件，它用于IP 协议中发送控制消息。也就是说，ICMP 是依靠 IP 协议来完成信息发送的，它是 IP 的主要部分，但是从体系结构上来讲，它位于 IP 之上，因为 ICMP 报文是承载在 IP 分组中的，就和 TCP 与 UDP 报文段作为 IP 有效载荷被承载那样。

NAT
