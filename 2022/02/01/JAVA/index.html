位运算符

  • `>>(有符号右移)：用符号位补左边
  • `>>>(无符号右移)：左边用0补齐

JAVA中，if的判断条件必须是bool类型，不能像C一样用大于零的数字

&|^也可以对bool类型进行操作

类似与&&、||，但是&&和||有短路功能。

以&&为例，当&&左面的变量为false时，则必定为false，就不会运算&&右面的变量，而&没有短路功能，依旧会判断右面的变量。

类型转换

扩宽基本类型转换

窄化基本类型转换

bool类型无法与其他类型进行数据转换

一元数字提升

  • 将byte、short、char类型的一元数字自动提升为int类型（扩宽基本类型转换）
  • 执行一元数字提升的情况
    • 数组的索引、创建数组时的数组长度
    • 一元运算符+-（正负号）
    • 按位取反（~）
    • 位移（<<、>>、>>>）

二元数字提升

两个不同类型的数字，通过扩宽基本类型转换，将范围小的数字类型转换为范围大的数字类型。

当两边的数字类型都不是double、float、long时，两个数字都转换成int类型

二元数字提升的情况

  • 乘（*）除（/）取余(%)
  • 加法、减法
  • 比较
  • 判断
  • 位运算（&、^、|）
  • 三目（？：）

关键字、标识符

关键字

标识符

标识符：变量名、方法名、类名等，命名给则如下

  1. 不限长度的JAVA字母、JAVA数字序列，但必须以JAVA字母开头（区分大小写）
  2. 不能使用关键字
  3. 不能使用字面量true、false、null

命名建议

变量名、方法名：小驼峰，myName

类名：大驼峰，MyName

全局变量：MY_NAME
